## Build

### Prerequisites

Install MacTeX (macOS):

```bash
brew install --cask mactex
```

After installation, restart your terminal or add to PATH:

```bash
export PATH="/Library/TeX/texbin:$PATH"
```

### Compile

```bash
pdflatex resume_yenchih_liao.tex
```

Output: `resume_yenchih_liao.pdf`
